## ✨ المميزات الجديدة

### 📁 تشفير الملفات بالكامل
- ✅ **تشفير أي نوع ملف** (PDF, ZIP, صور, فيديو, مستندات)
- ✅ **فك تشفير الملفات** واستلامها بنفس الاسم
- ✅ **دعم حتى 20 ميجابايت** (حد التليجرام)
- ✅ **تشفير AES-256 مع PBKDF2**
- ✅ **ملح عشوائي لكل ملف** (16 بايت)

### 📝 تشفير النصوص
- تشفير AES-256 للنصوص
- تشفير Base64
- توليد هاش SHA256/MD5
